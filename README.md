# Insurance Claims Analytics Dashboard | Qlik Sense

## 📌 Project Overview

This project is an enterprise-style **Insurance Claims Analytics Dashboard** developed using **Qlik Sense**. The dashboard is designed to analyze insurance claims performance across different business dimensions such as claim type, channel, zone, division, branch, plan type, policy vintage, and financial timeline.

The project uses **synthetic/sample data** and is created for portfolio, learning, and demonstration purposes. It simulates a real-world insurance claim analytics scenario where business users can monitor claim settlement performance, outstanding claims, due claims, reinvestment insights, and year-over-year performance.

This project demonstrates end-to-end BI development skills including data modeling, Qlik scripting, KPI creation, dashboard design, set analysis, financial year logic, and interactive reporting.

---

## 🎯 Business Problem

Insurance companies process a large number of claims across different regions, branches, products, and claim categories. Business users need a centralized dashboard to monitor claim performance and identify pending, paid, outstanding, and due claims.

Without an interactive analytics solution, it becomes difficult to:

- Track claim settlement performance
- Monitor outstanding claims
- Compare current year and last year performance
- Identify claims pending due to missing customer details
- Track claims due for current and upcoming months
- Analyze performance across business hierarchy
- Make timely business decisions

This dashboard solves the problem by providing a structured and interactive claims analytics solution.

---

## ✅ Project Objective

The main objective of this project is to build an interactive Qlik Sense dashboard that helps users:

- Monitor key insurance claims KPIs
- Analyze paid, payable, and outstanding claims
- Track claim settlement ratio
- Compare current year and last year performance
- Monitor due claims for current and upcoming months
- Analyze outstanding claims with missing customer details
- Track reinvestment consent and reinvestment amount
- View performance across Zone, Division, and Branch hierarchy
- Support business decision-making using visual analytics
